Pollen Machine

Overview

GoalDesign an interactive system that visualizes pollination and air-quality data to raise awareness about environmental health.
My RoleHardware prototyping, data visualization design, UX flow, and interactive installation setup.
TeamIndividual project guided by faculty and peer critique sessions.
DeliverablesArduino-based sensor prototype, live data dashboard, and exhibition-ready installation.

Responsibilities

  • Build a prototype device combining particle sensors and environmental data APIs.
  • Design a data visualization interface showing pollen density and air-quality variations.
  • Develop user interaction flow for museum or gallery installations.
  • Document sensor calibration, real-time feedback, and user reactions during testing.

Process

  • Research → Investigated urban air-quality tools and pollen-tracking devices.
  • Prototyping → Built sensor module using Arduino and PM2.5 sensor integration.
  • Data Visualization → Created simple dashboard in Processing to map real-time readings.
  • Exhibition → Installed system in gallery, observing user curiosity and engagement.
